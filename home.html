¿Qué voy a aprender?

A lo largo de este curso, Aprenderás a Programar una Aplicación Web increíble. Concretamente, ¡vas a crear la aplicación que esta viendo! Ademas, al acabar el curso tendrás la base de conocimiento suficiente para poder crear tus propias Aplicaciones Web.

Boostrap 4

Gracias a Bootstrap 4 aprenderás a crear diseños modernos y profesionales usando HTML5, CSS3 y JavaScript.

CodeIgniter
